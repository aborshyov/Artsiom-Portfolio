# Artsiom Barshchou Portfolio Website

This is a static portfolio website built from Artsiom's resume, selected photos, and AthleteOS project screenshot.

## Files
- `index.html` — main website page
- `styles.css` — styling
- `assets/Artsiom_Barshchou_Resume.pdf` — resume download file
- selected portfolio photos inside `assets/`
- `assets/athleteos-screenshot.jpg` — AthleteOS product screenshot used in the featured project section

## Added sections
- Target roles line
- Proof points
- AthleteOS case study: Problem → What I built → Tools used → Result
- Why Me section

## How to preview
Open `index.html` in your browser.

## How to deploy on Vercel
1. Create a new GitHub repository.
2. Upload these files.
3. Go to Vercel and import the repository.
4. Deploy as a static site.

## Free domain option
You can deploy for free using a Vercel subdomain like `yourname.vercel.app`, or GitHub Pages with a `github.io` URL.
A custom domain like `artsiombarshchou.com` usually costs money every year.
